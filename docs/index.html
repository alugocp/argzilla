JSON Config Reference

You'll want to maintain an argzilla.json file to use the tool. This section defines the different options available in your config file.

language (required)

Define the default language for the generated code. Set to either java, python, ruby, js, cpp, bash, lua.

out (required)

Define the default file to write the generated code to.

parameters

A JSON object with optional min and max integers that represent the minimum and maximum number of arguments (respectively). If you omit either field, it means you can have infinite arguments (if you omit max) or as few as zero (if you omit min). The minimum and maximum values don't count commands, flags or options.

"parameters":{
 "min":1,
 "max":2
}

flags

Flags are strings that your command line tool recognizes as keywords anywhere in the arguments list. Once your parameters are parsed, you can check whether or not these flags were given. They are essentially a boolean value (whether or not it exists). Flags all have a label (the reference to this flag as it exists in the code), and one or more names (the strings that your tool maps to a certain label).

"flags":[
 {
  "label":"string",
  "names":["string1","string2",...],
 },
 ...
]

options

Options are keywords that are given a string value in the command line. Think flags, but instead of a boolen value (whether or not it exists) it's given a string value (the argument passed right after the option keyword). Options have the same label and names structure as flags.

"options":[
 {
  "label":"string",
  "names":["string1","string2",...],
 },
 ...
]

commands

A command is a keyword that comes at the beginning of your arguments list. It can tell your tool what the user wants to do. It also allows you to define different flags and options fields for each command.

"commands":{
 "string":{
  "flags":[...],
  "options":[...],
 },
 ...
}

Generated Interfaces

Here we've documented the code generated in each language by Argzilla.

C++

ArgBox argparse(int argc,char** argv);

typedef struct {
 char* command=NULL or some string;
 Options options;
 Flags flags;
 std::vector<char*> args;
} ArgBox;

typedef struct {
 bools
} Flags;

typedef struct {
 char pointers
} Options;

Python

def argparse()

{
 "command":None or some string,
 "options":{ strings },
 "flags":{ booleans },
 "args":[ strings ]
}

Ruby

def argparse()

{
 :command => nil or some string,
 :options => { strings },
 :flags => { booleans },
 :args => [ strings ]
}

Node.js

function argparse();

{
 command:undefined or some string,
 options:{ strings },
 flags:{ booleans },
 args:[ strings ]
}

Bash

function argparse()

argbox_command=0 or some string
declare -A argbox_options # associative array with string values
declare -A argbox_flags # associative array with 0 or 1 value
argbox_args=( strings )

Lua

function argparse()

{
 command=nil or some string,
 options={ string table },
 flags={ boolean table },
 args={ string array }
}
