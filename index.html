What is Argzilla?

Argzilla is a simple command line tool that generates argparse functions for your programs. It is inspired by partial evaluators, which customize source code based on a config file to remove unnecessary portions. Argparsing is when you interpret the arguments passed into a command line tool, most commonly as flags (i.e. -a, --help, etc) or parameters (-L hello, for example). With Argzilla you can define your expected arguments in a JSON config file and then generate code to parse these patterns. Argzilla outputs code in Java, Ruby, Python, Lua, Bash, Node.js and C++.

This is an open source project developed by LugoCorp. If you find bugs or have any questions, just contact us below.
